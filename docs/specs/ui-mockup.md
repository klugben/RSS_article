## 格式规范

| 元素 | 规范 | 示例 |
|------|------|------|
| 标题行 | `📰 RSS 每日简报 — {订阅源名称}` | 固定前缀 + 来源 |
| 日期 | `YYYY-MM-DD` | 2026-04-06 |
| 主题标题 | `## {主题名}` | ## AI与智能体 |
| 文章条目 | `**{标题}**：{摘要}` | 加粗标题 + 冒号 + 摘要 |
| 原文链接 | `[原文]({url})` | Markdown 链接 |
| 要点概括 | `### 本日要点` + 2-3 句话 | 仅 >3 篇时使用 |
| 链接列表 | `1. **{标题}** ({主题}) — [原文]` | 编号列表，仅 >3 篇时使用 |

## 不使用

- 不使用飞书消息卡片（富文本卡片需 Bot 权限，当前使用群消息即可）
- 不使用 @提及
- 不使用 emoji（📰 除外）
- 不使用引用块或折叠
