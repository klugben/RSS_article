## 关键发现

1. **核心用户**：AI/科技领域的个人用户，每日需要快速获取 RSS 订阅文章精华
2. **核心旅程**：早晨 8:00 在飞书群自动收到简报 → 2 分钟内扫完 → 选择性深读
3. **核心流程**：lumen 抓取 → feeds.db 存储 → Agent 读取+分类+摘要 → 飞书推送
4. **核心领域对象**：Feed → Article → Topic → Digest → Channel
5. **关键约束**：400 字硬限制、只读查询、无文章不推送
